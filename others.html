    • General Mission Analysis Tool

    Currently, I am learning about the General Mission Analysis Tool (GMAT) developed by NASA. GMAT, is widely used in industry and academia to design and analyze orbital trajectories. So far I am following the tutorials provided in the documentation to familiarize myself with the software.

    Compute Unified Device Architecture

    Compute Unified Device Architecture (CUDA), is a parallel computing language based in c/c++ developed by Nvidia. I created a simple particle simulator to trace the trajectory of protons and electrons under a constant magnetic field in a collisionless regime. The aim of this project was to introduce myself into this powerful parallel computing language and compare its performance with traditional CPU simulations.
